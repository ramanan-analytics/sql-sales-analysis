# Sales Performance Analysis using SQL

## Project Overview
This project focuses on analyzing sales performance data using SQL to
identify revenue trends, top-performing products, and regional insights.
The analysis is designed to answer real-world business questions and
support data-driven decision-making.

## Tools Used
- SQL (Oracle SQL / MySQL)
- Excel (for data validation)
- GitHub

## Dataset Description
The dataset contains transactional sales data including order details,
product information, categories, regions, and revenue metrics.

## Key Business Questions
- How does revenue trend month-over-month?
- Which products generate the highest
